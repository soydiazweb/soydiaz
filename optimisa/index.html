Método principal

Test de resazurina para medir la velocidad de degradación de sacarosa.

Optimisa selecciona un indicador biológico que permite observar la velocidad con que ocurren las transformaciones y se degrada la sacarosa. Los metabolitos intermedios son de paso; los finales se acumulan o se evaporan parcialmente.

  • Indicador confiable para evaluar actividad microbiológica.
  • Resultado relativamente rápido y de bajo costo.
  • Viable para integrarse a un sistema de control continuo.
  • Facilita ecuaciones descriptivas y parámetros de conformidad.
  • Soporta decisiones oportunas para reducir el crecimiento microbiano.
